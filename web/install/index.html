How to Install the Autoplacer

Step 1: Install Tampermonkey

First install Tampermonkey, this is the extension in your browser that is going to run the script.

  • Chrome: Tampermonkey for Chrome
  • Firefox: Tampermonkey for Firefox
  • Opera: Tampermonkey for Opera
Step 2: Install the Script

Open this link: https://tubnetplace.live/scripts/tubnetplace.user.js

A Tampermonkey window will appear asking if you want to install the script, click INSTALL.

Step 3: Script Installation Complete

Now the script is installed! Head to https://www.reddit.com/r/place and see the magic happen!

The script sends notifications in the top-right corner.

You can check the stats and total placers on https://tubnetplace.live/

Step 4: Important Note

Remember to keep the Reddit Place tab open!!

Note: When a pixel is placed, the place button will still be orange. No panic, the pixel has been placed, the button just doesn't update.
